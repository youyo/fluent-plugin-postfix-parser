## Installation

### RubyGems

```
$ gem install fluent-plugin-postfix-parser
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-postfix-parser"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format filter postfix-parser
```
